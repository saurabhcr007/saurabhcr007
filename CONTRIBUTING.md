# Contributing to saurabhcr007/saurabhcr007

Thank you for your interest in improving this GitHub profile! 🎉

This is a personal GitHub profile README repository for **Saurabh Chaudhary** — a Cloud & DevOps engineer. Contributions are welcome in the form of bug reports, improvement suggestions, and design ideas.

## What You Can Contribute

- 🐛 **Bug reports** — Broken badges, broken links, broken SVGs, markdown rendering issues
- 💡 **Improvement suggestions** — Better badge colors, section ideas, layout improvements
- 🔧 **Technical fixes** — GitHub Actions YAML improvements, workflow optimizations
- 📝 **Typos / grammar** — Any text corrections

## What NOT to Contribute

- Personal information changes (name, social links, projects) — these are owner-controlled
- Opinionated design overhauls without prior discussion
- Dependencies or package.json files (this is a pure markdown/SVG repository)

## How to Contribute

1. **Fork** this repository
2. **Create a branch** — `git checkout -b fix/broken-link-connect-section`
3. **Make your changes** — keep them focused and minimal
4. **Test locally** — preview the markdown in a GitHub-style renderer if possible
5. **Open a Pull Request** — use the PR template provided

## Code Standards

- Markdown: Follow [GitHub Flavored Markdown](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)
- Badges: Use `shields.io` with `style=for-the-badge`
- SVGs: Inline SVG preferred; external hosted SVG acceptable from trusted CDNs only
- GitHub Actions: Use pinned action versions (e.g., `@v4` not `@master`)

## Issue Reporting

Use the issue templates provided in `.github/ISSUE_TEMPLATE/` to report bugs or suggest features.
